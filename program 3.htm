NOTE: For years we used Handlebars for templating; not anymore, Handlebars broke our code. We now use Nunjucks. You may hear or see references to Handlebars in text or videos, but it will be Nunjucks that we should be using for our templates.

Your node.js program will require 3 pages: a welcome page, a data display page containing a HTML form, and a summary page. Each of the 3 pages will require an express-handlebars template that is to be substituted into the main layout. To get full credit, your program will also need the following features.

  1. a main.handlebars layout containing a Bootstrap nav bar in the header and a generic footer that will display in each of the 3 pages. The nav bar should contain a logo-image, and that image should be retrieved from the public/images folder using static files. You can get the logo image from the link near the end of the page, or you can make your own.
  2. the nav bar should contain 2 links, one to the welcome page, and one to the data-display page.
  3. the body of the welcome page should display the image of an airplane that you provide, originating in the public/images folder using static files. if the image requires attribution, include the info for where the image came from.
  4. the data display page should take the provided array of flight info objects (shown below), and using Nunjucks repetition methods, generate and display a Bootstrap html table of the data when the page loads.
  5. the data display page will also need an html form that uses the post method. The form should have 2 text boxes, a drop down list with 3 choices, a set of 3 radio button each with the same name, and a submit button. An image is shown below.
  6. The form tag containing the submit button should have an action attribute that points to a route in index.js that retrieves the form data, then displays a summary of that data on the summary page.
  7. The main.handlebars template should have a {{page_title}} thing so that we can substitute the right title for each of the pages we display. So the welcome page should display the proper title to the user, the data-display page should display a proper page title, and so on.

Here's your list of files and folders for this project.

  • you'll need to install express, body-parser, and express-handlebars.
  • you'll need 3 regular templates in the views folder and a master template in the layouts folder.
  • a public folder with an images subfolder to hold your airplane pic and the logo
  • an index.js file, and a package.json file.
